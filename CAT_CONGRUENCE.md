# CAT Congruence – ScaffyLads

This project sits inside the broader **Coastal Alpine Tech** ecosystem.

## Alignment Rules (Mandatory for every build)

1. **Sovereign by default**  
   Local-first storage. Cloud sync is opt-in. No silent data leaving the device or NZ region without explicit consent.

2. **Reuse the wheel**  
   Frontend shell, Tauri desktop packaging, design tokens, auth patterns, and CI discipline are deliberately congruent with Front_Line_Whanau.

3. **Python for the brain**  
   The intelligent journal layer (voice, structured extraction, natural language queries, future local LLM) is Python/FastAPI. This matches Aether, Weaver, Byte Size Kai, and the wider edge stack.

4. **HITL for high-stakes**  
   Changes to compliance templates, billing, multi-user permissions, or production data migrations require human approval.

5. **Edge-ready**  
   Architecture leaves a clean path for later running inference or storage on CAT edge nodes (RPi 5 + Hailo class hardware).

6. **Te Mana Raraunga spirit**  
   Treat user data as taonga: owner-controlled, purpose-limited, and auditable.

## Non-Goals (for now)

- Full SiteGuard DAO / blockchain immutability
- Heavy multi-tenant enterprise features before product-market fit
- Training foundation models on user journals

## Relationship to Other CAT Projects

- **Front_Line_Whanau** → UI, Tauri, Supabase, and engineering patterns source
- **Aether / Weaver** → Future agent orchestration patterns
- **Byte Size Kai / SoilGuard / AquaGuard** → Shared sovereign edge philosophy
