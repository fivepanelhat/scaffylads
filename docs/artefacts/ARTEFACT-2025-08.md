### Why this document is the primary August artefact

This report consolidates the mid-2025 R&D story into a single investment-portfolio frame:

- **AgriTech mission** — “Operating System for Modern Agriculture,” targeting fragmented data silos, operational inefficiency, and supply-chain opacity.
- **Microgreens go-to-market** — **MicrogreensDAO** as the vertical beachhead brand (Taranaki-focused early adoption), with an underlying horizontal white-label platform (“Deepsearch” IP designation in the source).
- **Web3 / tokenisation roadmap** — Real-World Asset (**RWA**) **tokenisation** on the **Sui** blockchain, with **Walrus** decentralised storage and **DeepBook** CLOB called out as part of a differentiated competitive design.
- **Hybrid commercial model** — Production-oriented SaaS maturity claims (mobile, API, cloud IaC) paired with a longer-horizon decentralised marketplace vision.

The source is suitable as the **primary** August citation because it explicitly binds AgriTech product strategy, MicrogreensDAO GTM, and Sui/Web3 tokenisation into one R&D narrative dated in August 2025.

### Grounded research themes (from the primary source)

1. **AgriTech platform thesis** — Unified operations software for agricultural SMEs, not a single-feature toolset.  
2. **MicrogreensDAO as beachhead** — Relatable vertical brand for early clients; horizontal platform positioned for later multi-vertical expansion (e.g. real estate, art, carbon credits).  
3. **Web3 RWA tokenisation** — Strategic roadmap beyond subscription SaaS into transaction-based marketplace economics.  
4. **Sui ecosystem choices** — Sui L1 + Walrus storage + DeepBook liquidity layer as documented design intent.  
5. **Enterprise SaaS stack themes** — Cross-platform mobile (React Native / TypeScript), Python API, PostgreSQL / Redis, AWS + Kubernetes / Terraform, blue-green deployment, multi-layer security posture, compliance *readiness* language (GDPR / SOC 2 / HIPAA readiness — alignment, not claimed certification).

> **Grounding rule:** Valuation ranges, fundraising targets, and commercial forecasts appear in the source as *analytical estimates*. This artefact does **not** republish those figures as market facts. They are acknowledged only as internal R&D modelling of the time.
