### Grounded research themes (from the primary source)

#### Architectural tiers (Gold / Diamond / Platinum Edge)

For each vertical, the blueprint repeats a three-layer manufacturing pattern:

1. **Gold Standard — Workflow-native design**  
   End-to-end process phases mapped as digital modules (input → deliberation/monitoring → decision/response → execution → archive/audit).

2. **Diamond Standard — Enterprise foundation**  
   Hybrid infrastructure (cloud control plane + on-prem / edge data plane), security patterns, and **Web3 trust** components (e.g. Sui Walrus, on-chain hashes, object-centric asset models).

3. **Platinum Edge — Agentic architecture**  
   Local agent orchestration (e.g. LangGraph / NeMo Guardrails patterns), specialised agent roles, and edge inference on **DGX Spark**.

#### Vertical applications documented (research designs)

| Vertical (as titled in source) | Domain focus | Illustrative Gold workflow | Edge / trust notes |
| --- | --- | --- | --- |
| **Rangatiratanga DAO** | Governance & sovereignty OS | Hui → Wānanga → Whakatau → Mahere → Pūrākau | On-prem data plane; Sui Walrus for records; governance token concept |
| **SiteGuard DAO** | Construction & compliance OS | Induction → monitoring → incident → audit → clearance | Edge-heavy DGX on site; Sui hashes of safety logs |
| **GridWatch DAO** | Critical infrastructure security | Baseline → anomaly → isolation → forensics → recovery | Air-gap capable edge; on-chain event hashes |
| **Origin DAO** | Supply chain & provenance OS | Genesis twin → journey → custody handshake → market → consumption | **Tokenisation** of assets; **DeepBook** for forward markets; Sui objects |
| **TradesPassport DAO** | Credential & identity OS | Verify → mint → deploy → upskill → reputation | Soulbound / portable credentials; ZKP privacy patterns; Sui kiosk primitives |

#### Tokenisation, Sui, and edge (as written)

- **Sui** is the trust / ledger layer for immutable decision trails, safety-log integrity, provenance objects, and credential primitives.  
- **Tokenisation** appears especially in **Origin DAO** (digital twins of high-value exports, pre-sale / financing tokens, marketplace bidding) and identity minting in **TradesPassport**.  
- **Edge** compute is not optional decoration: DGX Spark is specified as the sovereign / on-site intelligence plane for latency, privacy, and offline-capable operation.

> **Grounding rule:** Verticals above are recorded as **architectural research designs** in the source, not as claims that each product was fully shipped to production by December 2025.
